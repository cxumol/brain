The **sigmoid function** is defined as:

$$\sigma(x) = \frac{1}{1 + e^{-x}} = \frac{e^x}{e^x + 1}$$ ^88a827

It is often useful as a way of mapping the whole real number line to a number between $0$ and $1$.
