brain/Clojure map

Maps are Clojure's type of Associative array

  • A map is created in Clojure using the following syntax:

    • clojure (def my-map {:value-1 1 :value-2 2 :value-3 3})



      • The keywords don't need to be a keyword type, but this is convention.


      • There's also no need to have the entries on separate lines, but this increases readability.



  • Adding values to an existing map is done via the assoc function

    • clojure user=> (assoc my-map :value-4 4) {:value-1 1, :value-2 2, :value-3 3, :value-4 4}

  • Removing values from an existing map is done via the dissoc function

    • clojure user=> (dissoc my-map :value-2) {:value-1 1, :value-3 3}

  • Looking up a key is done two ways:

    • Either via the get function:



      • clojure user=> (get my-map :value-1) 1



    • or via invoking the map itself:



      • clojure user=> (my-map :value-1) 1


      • This is sensitive to null values





        • Example:







          • ```clojure




            user=> (def bad-lookup-map nil)
