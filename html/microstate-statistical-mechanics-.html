A microstate is a specific configuration of a thermodynamical system at the microscopic level. This is in contrast to a macrostate, which refers to properties like temperature, pressure, volume or density. We can often specify a microstate as the vector of the state of each of the $N$ elements in the system $(q_1,...,q_N)$, where the $q_i$'s are the properties measured for the $i$th element.
